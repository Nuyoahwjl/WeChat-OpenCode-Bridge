// core features

核心功能

🔗
无缝桥接

通过 WeChat ilink API 连接个人账号，长轮询实时监听消息，实现与 OpenCode AI 的双向通信。

💬
AI 自动回复

发送任意消息，AI 自动调用 OpenCode REST API 生成回复，支持流式输出，响应快速自然。

🔄
多会话管理

创建、重命名、删除、切换会话，聊天历史本地持久化存储，随时恢复上下文。

📱
扫码极速绑定

首次绑定只需在终端中用微信扫描二维码，凭证本地安全保存，后续自动登录。

⚡
斜杠指令系统

内置 7 种斜杠指令路由，通过微信消息即可管理会话、查看历史、获取帮助，完全无需额外 UI。

🛡️
健壮的错误处理

内置 fetch 重试机制、超时保护、结构化日志、会话状态追踪（idle/processing），稳定可靠运行。

🔌 wechat/ 模块

通过 WeChat ilink API 实现二维码登录、消息接收与发送，内置 fetch 包装器提供重试与超时保护，长轮询监听器保持消息实时推送。

🤖 opencode/ 模块

封装 OpenCode REST API 客户端，负责会话的创建、查询、删除与消息发送，对接本地运行的 opencode serve 服务。

💾 store/ 模块

账号凭证与会话数据持久化，JSON 文件存储于用户主目录下，聊天历史本地备份，会话状态追踪（idle / processing）。

⚡ commands/ 路由

斜杠指令解析与分发，将微信消息中的 /help、/new、/switch 等指令路由到对应处理函数，支持模糊匹配与参数解析。

// slash commands

微信快捷指令

在微信中发送以下指令，即可完整管理 OpenCode 会话，无需任何额外界面。

指令 说明 示例
/help 显示所有可用指令的帮助信息 /help
/new [标题] 创建新会话，有参数则以标题命名，无参数使用时间戳 /new 我的项目
/rename <标题> 重命名当前会话，自动添加 WeChat: 前缀 /rename 新名字
/delete [标题] 删除指定会话（含关键词匹配），无参数则删除当前会话 /delete 项目
/history [数量] 查看聊天历史，默认最近 20 条 /history 10
/sessions 列出 OpenCode 服务器上所有会话，显示 ID、时间与当前标记 /sessions
/switch <标题> 模糊匹配并切换会话，自动恢复本地历史记录 /switch 项目

五步快速上手

01
📦
克隆仓库

将项目克隆到本地并进入目录。

git clone https://github.com/Nuyoahwjl/WeChat-OpenCode-Bridge
02
⚙️
安装依赖

安装完成后自动编译 TypeScript。

npm install
03
📱
绑定微信

首次使用，扫码完成账号绑定，凭证自动保存。

npm run setup
04
🤖
启动 OpenCode

在你的工作目录下启动 OpenCode 服务。

opencode serve
05
🚀
启动桥接

回到仓库目录，启动桥接服务，开始对话！

npm run start
// npm scripts
命令 说明
npm run setup 扫码绑定微信（首次使用）
npm run start 启动桥接服务
npm run dev 开发模式（代码变更自动重新编译）
npm run build 手动编译 TypeScript
npm run status 显示当前绑定账号和会话状态
